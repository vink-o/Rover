# Mars Rover Simulation

A C++ console application that simulates the movement and navigation of a robotic rover on a 2D planetary grid. This project focuses on algorithmic movement and basic environmental interaction.

### Features
* **Grid Navigation:** Precise movement logic across a defined 2D coordinate system (X, Y).
* **Command Processing:** Sequence-based input handling for rover maneuvers (Forward, Backward, Left, Right).
* **Obstacle Awareness:** Logic for detecting and interacting with planetary terrain/boundaries.
* **Status Reporting:** Real-time feedback of the rover’s current position and orientation.

### Technical Details
* **Language:** C++
* **Logic:** 2D Array manipulation and coordinate transformation.
* **Focus:** Algorithmic efficiency and clean control flow.

### Implementation
The simulation uses a structured approach to track the rover's state, ensuring that movements are validated against the grid's boundaries before execution.
